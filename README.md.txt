Первая задача:
Этот бот умеет выбирать фильм по жанрам и годам, заданными пользователем, основываясь на данных сайта kinopoisk.ru.
Команды для общения:
/start - запускает бота 
/help - вызов справки 
/restart - перезапуск бота 
/genres - задать жанры 
/years - задать года 
/movie - найти фильм 
/stop - закрыть программу
